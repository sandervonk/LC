Q1: Vector - Components

Question:

The components of a vector V⃗\vec{V}V are often written as (VxV_xVx​, VyV_yVy​, VzV_zVz​). What is the y component of a vector which is the sum of the two vectors, V1⃗\vec{V_1}V1​​ and V2⃗\vec{V_2}V2​​, whose components are (3.09, 1.83, 0.00) and (3.66, -4.49, -2.73)?
Calculate the length of the sum of [(V)\vec]1 and [(V)\vec]2.

Variable Breakdown

V1⃗=(3.09,1.83,0.00)\vec{V_1} = (3.09, 1.83, 0.00)V1​​=(3.09,1.83,0.00)
V2⃗=(3.66,−4.49,−2.73)\vec{V_2} = (3.66, -4.49, -2.73)V2​​=(3.66,−4.49,−2.73)

Steps / Summary

Part 1: finding a component

To combine these two vectors, we simply add their directional magnitude scalar components:

Vsum⃗=V1+2⃗\vec{V_{sum}} = \vec{V_{1+2}}Vsum​​=V1+2​​
Vsum⃗=(V(1+2),x,V(1+2)y,V(1+2),z)\vec{V_{sum}} = (V_{(1+2),x}, V_{(1+2)y}, V_{(1+2),z})Vsum​​=(V(1+2),x​,V(1+2)y​,V(1+2),z​)
Vsum⃗=(V1,x+V2,x,V1,y+V2,y,V1,z+V2,z)\vec{V_{sum}} = (V_{1,x} + V_{2,x}, V_{1,y} + V_{2,y}, V_{1,z} + V_{2,z})Vsum​​=(V1,x​+V2,x​,V1,y​+V2,y​,V1,z​+V2,z​)

Though we won’t need all these components for the start of this question, it’s worth doing a little more work to make sure we understand it!

For my variation / numbers, I’ll get a Vsum⃗\vec{V_{sum}}Vsum​​ of:

Vsum⃗=(3.09+3.66,1.83+−4.49,0.00+−2.73)\vec{V_{sum}} = (3.09 + 3.66, 1.83 + -4.49, 0.00 + -2.73)Vsum​​=(3.09+3.66,1.83+−4.49,0.00+−2.73)
Vsum⃗=(6.75,−2.66,−2.73)\vec{V_{sum}} = (6.75, -2.66, -2.73)Vsum​​=(6.75,−2.66,−2.73)

Since my example prompt asks for the y-component, i’ll enter -2.66

Part 2: Finding vector length

To find the length of the sum vector Vsum⃗\vec{V_{sum}}Vsum​​ we found, we can use the Pythagorean theorem in three dimensions:

d=(Δx)2+(Δy)2+(Δz)2d = \sqrt{\left( \Delta x \right)^2 + \left( \Delta y \right)^2 + \left( \Delta z \right)^2}d=(Δx)2+(Δy)2+(Δz)2​

For a vector, this will work out to be:

d=(Vx⃗)2+(Vy⃗)2+(Vz⃗)2d = \sqrt{\left( \vec{V_x} \right)^2 + \left( \vec{V_y} \right)^2 + \left( \vec{V_z} \right)^2}d=(Vx​​)2+(Vy​​)2+(Vz​​)2​

or more technically:

d=(Vsum,x⃗)2+(Vsum,y⃗)2+(Vsum,z⃗)2d = \sqrt{\left( \vec{V_{sum,x}} \right)^2 + \left( \vec{V_{sum,y}} \right)^2 + \left( \vec{V_{sum,z}} \right)^2}d=(Vsum,x​​)2+(Vsum,y​​)2+(Vsum,z​​)2​

For my example, this would be

d=(6.75)2+(−2.66)2+(−2.73)2d = \sqrt{(6.75)^2 + (-2.66)^2 + (-2.73)^2}d=(6.75)2+(−2.66)2+(−2.73)2​
d=7.75d = 7.75d=7.75

For a final solution of 7.75

Q2: Vector - Magnitude

Question:

If VxV_xVx​ = 19.00 units and VyV_yVy​ = -15.10 units, determine the magnitude of V⃗\vec{V}V.
Determine the angle of V⃗\vec{V}V with respect to the x-axis. (in deg)

Variable Breakdown

Vx⃗=19.00\vec{V_x} = 19.00Vx​​=19.00
Vy⃗=−15.10\vec{V_y} = -15.10Vy​​=−15.10

Steps / Summary

Part 1: Magnitude

We can use the Pythagorean theorem to solve for the magnitude:

a2+b2=c2a^2 + b^2 = c^2a2+b2=c2
(Vx)2+(Vy)2=V2=∣V⃗∣2\left(V_x\right)^2 +\left(V_y\right)^2 = V^2 = |\vec{V}|^2(Vx​)2+(Vy​)2=V2=∣V∣2
∣V⃗∣2=(Vx)2+(Vy)2|\vec{V}|^2 = \left(V_x\right)^2 +\left(V_y\right)^2∣V∣2=(Vx​)2+(Vy​)2
∣V⃗∣=(Vx)2+(Vy)2|\vec{V}| = \sqrt{\left(V_x\right)^2 +\left(V_y\right)^2}∣V∣=(Vx​)2+(Vy​)2​

With the numbers from my example, that would be:

∣V⃗∣=(19.00)2+(−15.10)2|\vec{V}| = \sqrt{\left(19.00\right)^2 +\left(-15.10\right)^2}∣V∣=(19.00)2+(−15.10)2​
∣V⃗∣=24.27|\vec{V}| = 24.27∣V∣=24.27

Giving me a solution for this part of 24.27

Part 2: Angle

To find the angle of Vsum⃗\vec{V_{sum}}Vsum​​ relative to the x-axis, we can do a bit of sohcahtoa

note that for different VxV_xVx​ and VyV_yVy​ values you may have to adjust the calculated angle in order to account for the bounds of the inverse trigonometric functions. If this is the case, your new angle should be a multiple of 90 degrees off from your calculations.

Using the diagram above, we can conclude that to find the angle θ\thetaθ, we may use the relationship:

θ=tan⁡(oppositeadjacent)−1\theta = \tan{\left( \frac{\text{opposite}}{\text{adjacent}} \right)}^{-1}θ=tan(adjacentopposite​)−1

Q3: 2D Kinematics - Displacement

Question:

A delivery truck travels 11 km north, 19 km east, and 28 km south. What is its final displacement from the origin? (in km)
What is the angle of the final displacement relative to the east?

Variable Breakdown

Vx=11 km+−28 kmV_x = 11 \ \text{km} + -28 \ \text{km}Vx​=11 km+−28 km
Vx=−17 kmV_x = -17 \ \text{km}Vx​=−17 km
Vy=19 km+0 kmV_y = 19 \ \text{km} + 0 \ \text{km}Vy​=19 km+0 km
Vy=19 kmV_y = 19 \ \text{km}Vy​=19 km

Steps / Summary

Keeping in mind the following relationships of compass headings to axes and signs, add together the components for your VxV_xVx​ and VyV_yVy​, as shown in the Variable Breakdown above:

North →\rightarrow→ +Vy+V_y+Vy​
South →\rightarrow→ −Vy-V_y−Vy​
East →\rightarrow→ +Vx+V_x+Vx​
West →\rightarrow→ −Vx-V_x−Vx​

Now that you have your VxV_xVx​ and VyV_yVy​, we can begin to solve the problem.

Part 1: Distance (position vector magnitude)

To find the final displacement, we’re just finding the distance travelled, which is the length/magnitude ∣V⃗∣|\vec{V}|∣V∣.

Just like in previous problems, we can find the magnitude from the VxV_xVx​ and VyV_yVy​ components using the Pythagorean theorem (though this time just in two dimensions)

Adapting the Pythagorean theorem for vector magnitude:

a2+b2=c2a^2 + b^2 = c^2a2+b2=c2
(Vx)2+(Vy)2=V2=∣V⃗∣2\left(V_x\right)^2 +\left(V_y\right)^2 = V^2 = |\vec{V}|^2(Vx​)2+(Vy​)2=V2=∣V∣2
∣V⃗∣2=(Vx)2+(Vy)2|\vec{V}|^2 = \left(V_x\right)^2 +\left(V_y\right)^2∣V∣2=(Vx​)2+(Vy​)2
∣V⃗∣=(Vx)2+(Vy)2|\vec{V}| = \sqrt{\left(V_x\right)^2 +\left(V_y\right)^2}∣V∣=(Vx​)2+(Vy​)2​

With the numbers from my example, that would be:

∣V⃗∣=(−17 km)2+(19 km)2|\vec{V}| = \sqrt{\left(-17 \ \text{km}\right)^2 +\left(19 \ \text{km}\right)^2}∣V∣=(−17 km)2+(19 km)2​
∣V⃗∣=25.50 km|\vec{V}| = 25.50 \ \text{km}∣V∣=25.50 km

Giving me a solution for this part of 25.50 km

Part 2: Angle

To again find the angle of the final displacement, we can use the same process as Part 2 of Q2: trigonometrically

Reusing the equation:

θ=tan⁡(VyVx)−1\theta = \tan{\left( \frac{V_y}{V_x} \right)}^{-1}θ=tan(Vx​Vy​​)−1

For my example scalar VxV_xVx​ and VyV_yVy​ components:

θ=tan⁡(−17 km19 km)−1\theta = \tan{\left( \frac{-17 \ \text{km}}{19 \ \text{km}} \right)}^{-1}θ=tan(19 km−17 km​)−1
θ=−41.82 deg\theta = -41.82 \ \text{deg}θ=−41.82 deg

We can again check that this is in the correct quadrant. a positive VxV_xVx​ and negative VyV_yVy​ should land us in the forth quadrant, which lines up again with our result being in the range (−90 deg≤θ≤0 deg)\left( -90 \ \text{deg} \le \theta \le 0 \ \text{deg} \right)(−90 deg≤θ≤0 deg)

This means that the final answer in my case is simply -41.82 deg
