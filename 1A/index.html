Q1: Vector - Components

Question:

The components of a vector V⃗\vec{V}V are often written as (VxV_xVx​, VyV_yVy​, VzV_zVz​). What is the y component of a vector which is the sum of the two vectors, V1⃗\vec{V_1}V1​​ and V2⃗\vec{V_2}V2​​, whose components are (3.09, 1.83, 0.00) and (3.66, -4.49, -2.73)?

Vsum⃗=(V1,x+V2,x,V1,y+V2,y,V1,z+V2,z)\vec{V_{sum}} = (V_{1,x} + V_{2,x}, V_{1,y} + V_{2,y}, V_{1,z} + V_{2,z})Vsum​​=(V1,x​+V2,x​,V1,y​+V2,y​,V1,z​+V2,z​)

Just like in previous problems, we can find the magnitude from the VxV_xVx​ and VyV_yVy​ components using the Pythagorean theorem (though this time just in two dimensions)

To find the dot product of vectors, we can use the following expression:

V1,y = ∣V1⃗∣⋅sin⁡θV_{1,y} \ = \ |\vec{V_1}| \cdot \sin{\theta}V1,y​ = ∣V1​​∣⋅sinθ

Q8: Cross Product 2

Question

A⃗\vec{A}A has a magnitude of 151515 and a direction of 110°110°110° in the x-y plane. B⃗\vec{B}B has a magnitude of 555 and a direction of 280°280°280° in the xxx-yyy plane. What is the direction of A⃗×B⃗\vec{A} \times \vec{B}A×B?
[choose from a list of +/−+/-+/− xxx, yyy, and zzz directions]

Variable Breakdown

∣A⃗∣=15|\vec{A}| = 15∣A∣=15
∣B⃗∣=5|\vec{B}| = 5∣B∣=5
θA=110°\theta_A = 110°θA​=110°
θB=280°\theta_B = 280°θB​=280°

Steps / Solution

Cross products are a pretty weird concept to get ahold of. Ultimately, they’re most interesting when we consider their geometric value as the vector perpendicular to its two ‘factors’.

I’d strongly recommend this Khan Academy article if you’re still feeling stuck. Some key takeaways from the article are:

  • The magnitude of the cross product is similar to the value of the dot product, being ∣A⃗∣∣B⃗∣sin⁡θ|\vec{A}||\vec{B}|\sin{\theta}∣A∣∣B∣sinθ to the dot product’s ∣A⃗∣∣B⃗∣cos⁡θ|\vec{A}||\vec{B}|\cos{\theta}∣A∣∣B∣cosθ.
  • We can use the right-hand rule, or draw a parallelogram, in order to find the direction perpendicular to both vectors.
  • While dot products work in 2 dimensions, cross products are only consistent in 3.

We’re going to do the math and write out the full cross product for the sake of completeness. If you want to preserve your sanity however, I’d suggest either drawing this one out, or using the right-hand rule when visualizing it. If you end up struggling on the next problem, in which you will be forced to use this method, you can always come back!

The Matrix

The basic representation of the matrix we’ll be using for the derivative shortcut is as follows:

[i^j^k^AxAyAzBxByBz]\begin{bmatrix} \hat{i} & \hat{j} & \hat{k} \\ A_x & A_y & A_z \\ B_x & B_y & B_z \end{bmatrix}⎣⎡​i^Ax​Bx​​j^​Ay​By​​k^Az​Bz​​⎦⎤​

The Determinant

Using the steps outlined in the associated Khan Academy article we can find the determinant:

A⃗×B⃗ = i^det⁡[AyAzByBz]−j^det⁡[AxAzBxBz]+k^det⁡[AxAyBxBy]\vec{A} \times \vec{B} \ = \ \hat{i}\det{\begin{bmatrix} A_y & A_z \\ B_y & B_z \end{bmatrix}} - \hat{j}\det{\begin{bmatrix} A_x & A_z \\ B_x & B_z \end{bmatrix}} + \hat{k}\det{\begin{bmatrix} A_x & A_y \\ B_x & B_y \end{bmatrix}}A×B = i^det[Ay​By​​Az​Bz​​]−j^​det[Ax​Bx​​Az​Bz​​]+k^det[Ax​Bx​​Ay​By​​]
A⃗×B⃗ = i^(AyBz−AzBy)−j^(AxBz−AzBx)+k^(AxBy−AyBx)\vec{A} \times \vec{B} \ = \ \hat{i}\left( A_y B_z - A_z B_y \right) - \hat{j}\left( A_x B_z - A_z B_x \right) + \hat{k}\left( A_x B_y - A_y B_x \right)A×B = i^(Ay​Bz​−Az​By​)−j^​(Ax​Bz​−Az​Bx​)+k^(Ax​By​−Ay​Bx​)

Putting it all together

There’s no easy way to show this generally without ending up with an overly verbose matrix, so pardon the mess. I’ll skip over a lot for this question, so if you have any questions afterward, just shoot me a message and I’ll try my best to explain things better in the context of an actual problem. In the meantime however, we’ll just use the values I got, though them all being on the same xxx-yyy plane doesn’t really help us understand how this process really works.

I’ll be skipping over the reasoning behind the trig but in this case the matrix is roughly equal to:

[i^j^k^∣A⃗∣cos⁡θA∣A⃗∣sin⁡θA0∣B⃗∣cos⁡θB∣B⃗∣sin⁡θB0]\begin{bmatrix} \hat{i} & \hat{j} & \hat{k} \\ |\vec{A}|\cos{\theta_A} & |\vec{A}|\sin{\theta_A} & 0 \\ |\vec{B}|\cos{\theta_B} & |\vec{B}|\sin{\theta_B} & 0 \end{bmatrix}⎣⎡​i^∣A∣cosθA​∣B∣cosθB​​j^​∣A∣sinθA​∣B∣sinθB​​k^00​⎦⎤​

From which we can calculate the determinant (cross product) as follows:

A⃗×B⃗ = i^(0−0)−j^(0−0)+k^((∣A⃗∣cos⁡θA)(∣B⃗∣sin⁡θB)−(∣A⃗∣sin⁡θA)(∣B⃗∣cos⁡θB))\vec{A} \times \vec{B} \ = \ \hat{i}\left( 0 - 0 \right) - \hat{j}\left( 0 - 0 \right) + \hat{k}\left( \left( |\vec{A}|\cos{\theta_A} \right)\left( |\vec{B}|\sin{\theta_B} \right) - \left( |\vec{A}|\sin{\theta_A} \right)\left( |\vec{B}|\cos{\theta_B} \right) \right)A×B = i^(0−0)−j^​(0−0)+k^((∣A∣cosθA​)(∣B∣sinθB​)−(∣A∣sinθA​)(∣B∣cosθB​))
A⃗×B⃗ = k^((∣A⃗∣cos⁡θA)(∣B⃗∣sin⁡θB)−(∣A⃗∣sin⁡θA)(∣B⃗∣cos⁡θB))\vec{A} \times \vec{B} \ = \ \hat{k}\left( \left( |\vec{A}|\cos{\theta_A} \right)\left( |\vec{B}|\sin{\theta_B} \right) - \left( |\vec{A}|\sin{\theta_A} \right)\left( |\vec{B}|\cos{\theta_B} \right) \right)A×B = k^((∣A∣cosθA​)(∣B∣sinθB​)−(∣A∣sinθA​)(∣B∣cosθB​))

While before this was still applicable to any vector set of A⃗\vec{A}A and B⃗\vec{B}B both existing in the xxx-yyy plane, from here on out this is only applicable to my specific vectors. Note that since it’s based on the unit vector k^\hat{k}k^, we already know that our result will be on the zzz-axis, we just need to find out if it’s positive or negative.

A⃗×B⃗ = k^((∣A⃗∣cos⁡θA)(∣B⃗∣sin⁡θB)−(∣A⃗∣sin⁡θA)(∣B⃗∣cos⁡θB))\vec{A} \times \vec{B} \ = \ \hat{k}\left( \left( |\vec{A}|\cos{\theta_A} \right)\left( |\vec{B}|\sin{\theta_B} \right) - \left( |\vec{A}|\sin{\theta_A} \right)\left( |\vec{B}|\cos{\theta_B} \right) \right)A×B = k^((∣A∣cosθA​)(∣B∣sinθB​)−(∣A∣sinθA​)(∣B∣cosθB​))
A⃗×B⃗ = k^((15cos⁡110°)(5sin⁡280°)−(15sin⁡110°)(5cos⁡280°))\vec{A} \times \vec{B} \ = \ \hat{k}\left( \left( 15\cos{110°} \right)\left( 5\sin{280°} \right) - \left( 15\sin{110°} \right)\left( 5\cos{280°} \right) \right)A×B = k^((15cos110°)(5sin280°)−(15sin110°)(5cos280°))
A⃗×B⃗ = k^(13.02)\vec{A} \times \vec{B} \ = \ \hat{k}\left( 13.02 \right)A×B = k^(13.02)
13.02 is positive, so the cross product follows the k^\hat{k}k^ unit vector with a positive coefficient.

Giving me a final vector direction along the positive zzz-axis?

“But wait”, you might say, “we calculated A⃗×B⃗\vec{A} \times \vec{B}A×B, but we need A⃗×B⃗\vec{A} \times \vec{B}A×B!”

You would of course be right, I did it wrong. Thankfully there’s a simple relationship between the two orderings of cross products:

B⃗×A⃗ = −(A⃗×B⃗)\vec{B} \times \vec{A}\ = \ -\left(\vec{A} \times \vec{B}\right)B×A = −(A×B)

I’m so glad I remembered to include that purposeful teaching moment (and definitely not mistake as a result of me mis-reading the question), since we now know that all we need to do is reverse the direction of the cross product we found in the incorrect order (only direction changes, not magnitude, though you could think of it being the inverse of the magnitude in the same direction as previously)

With this in mind, our real final answer is: the vector is in the negative z direction

Question

A⃗\vec{A}A has components <6.1, −5.1, 8.3>< 6.1 , \ -5.1 , \ 8.3 ><6.1, −5.1, 8.3>. B⃗\vec{B}B has components <−3.1, −5.5, 2.3>< -3.1 , \ -5.5 , \ 2.3 ><−3.1, −5.5, 2.3>. What is the y scalar component of A⃗×B⃗\vec{A} \times \vec{B}A×B ?

Variable Breakdown

A⃗ = <6.1, −5.1, 8.3>\vec{A} \ = \ < 6.1 , \ -5.1 , \ 8.3 >A = <6.1, −5.1, 8.3>
B⃗ = <−3.1, −5.5, 2.3>\vec{B} \ = \ < -3.1 , \ -5.5 , \ 2.3 >B = <−3.1, −5.5, 2.3>

Steps / Solution

Matrix

To find the cross product, we’ll set up the matrix in the format:

[i^j^k^AxAyAzBxByBz]\begin{bmatrix} \hat{i} & \hat{j} & \hat{k} \\ A_x & A_y & A_z \\ B_x & B_y & B_z \end{bmatrix}⎣⎡​i^Ax​Bx​​j^​Ay​By​​k^Az​Bz​​⎦⎤​

which has a determinant (cross product) of:

A⃗×B⃗ = i^(AyBz−AzBy)−j^(AxBz−AzBx)+k^(AxBy−AyBx)\vec{A} \times \vec{B} \ = \ \hat{i}\left( A_y B_z - A_z B_y \right) - \hat{j}\left( A_x B_z - A_z B_x \right) + \hat{k}\left( A_x B_y - A_y B_x \right)A×B = i^(Ay​Bz​−Az​By​)−j^​(Ax​Bz​−Az​Bx​)+k^(Ax​By​−Ay​Bx​)

Note: if you’re feeling lost on where this comes from, take a look at the cross product steps for Q8, especially where we use the det⁡\detdet in order to find the cross product.

Solving the problem

Since the question only asks for the scalar yyy-component, we can just take the part that’ll become the coefficient of the j^\hat{j}j^​ unit vector:

(A⃗×B⃗)y = −j^(AxBz−AzBx)\left(\vec{A} \times \vec{B}\right)_y \ = \ -\hat{j}\left( A_x B_z - A_z B_x \right)(A×B)y​ = −j^​(Ax​Bz​−Az​Bx​)

If you’re still feeling confused, I’d recommend this Khan Academy video, which gives a better overview of cross products.

For my example problem, this process works out to:

(A⃗×B⃗)y = −j^(AxBz−AzBx)\left(\vec{A} \times \vec{B}\right)_y \ = \ -\hat{j}\left(A_x B_z - A_z B_x\right)(A×B)y​ = −j^​(Ax​Bz​−Az​Bx​)
(A⃗×B⃗)y = −j^(6.1∗2.3−8.3∗−3.1)\left(\vec{A} \times \vec{B}\right)_y \ = \ -\hat{j}\left(6.1 * 2.3 - 8.3 * -3.1\right)(A×B)y​ = −j^​(6.1∗2.3−8.3∗−3.1)
(A⃗×B⃗)y = −j^(39.76)\left(\vec{A} \times \vec{B}\right)_y \ = \ -\hat{j}\left(39.76\right)(A×B)y​ = −j^​(39.76)
(A⃗×B⃗)y = −39.76∗j^\left(\vec{A} \times \vec{B}\right)_y \ = \ -39.76*\hat{j}(A×B)y​ = −39.76∗j^​

Giving me a final solution of -39.76
