Q1: Racecar Qualifying

Question:

Q2: Imperial Car Speed

Question:

The total distance treaveled by a car moving in a straight line is as follows:
  • After the first 7.0 min7.0 \ \text{min}7.0 min it has gone a total of 2.0 mi2.0 \ \text{mi}2.0 mi.
  • After 14.0 min14.0 \ \text{min}14.0 min it has traveled a total of 4.5 mi4.5 \ \text{mi}4.5 mi.
  • After 21.0 min21.0 \ \text{min}21.0 min it has traveled a total of 6.0 mi6.0 \ \text{mi}6.0 mi.
Find the average speed at:
  • Time = 7.0 min7.0 \ \text{min}7.0 min (in mihr\frac{\text{mi}}{\text{hr}}hrmi​)
  • Time = 14.0 min14.0 \ \text{min}14.0 min (in mihr\frac{\text{mi}}{\text{hr}}hrmi​)
  • Time = 21.0 min21.0 \ \text{min}21.0 min (in mihr\frac{\text{mi}}{\text{hr}}hrmi​)

Variable Breakdown

t1=7.0 mint_1 = 7.0 \ \text{min}t1​=7.0 min
d1=2.0 mid_1 = 2.0 \ \text{mi}d1​=2.0 mi
t2=14.0 mint_2 = 14.0 \ \text{min}t2​=14.0 min

t3=21.0 mint_3 = 21.0 \ \text{min}t3​=21.0 min
d3=6.0 mid_3 = 6.0 \ \text{mi}d3​=6.0 mi

Please note that in this case ddd represents distance, not displacement. Distance is total distance traveled, also represented by: ∫t0tf∣dx∣\int_{t_0}^{t_f}{|dx|}∫t0​tf​​∣dx∣ or ∫t0tf∣Δx∣\int_{t_0}^{t_f}{|\Delta x|}∫t0​tf​​∣Δx∣, though knowing what that means is not necessary to completing the problem.

We’ll also be using vvv to denote speed. Though semantially it is more correct to use a italic v for speed, and a boldface v for velocity, it’s hard to make that distinction with the current TeX system I’m using :(

Setup

To find the average speed vˉ\bar vvˉ of an object travelling a distance ddd over a time ttt, we can use the equation:

vˉ=dt\bar v = \frac{d}{t}vˉ=td​

Applied to this question, the process is fairly simple:

vˉn=dntn\bar v_n = \frac{d_n}{t_n}vˉn​=tn​dn​​

Solutions using this method can be found below for the timesteps t1t_1t1​, t2t_2t2​, and t3t_3t3​.

At the timestep t=t1t = t_1t=t1​

vˉ1=d1t1\bar v_1 = \frac{d_1}{t_1}vˉ1​=t1​d1​​

Which for my example problem evaluates to:

vˉ1=2.0 mi7.0 min∗60 min1 hr\bar v_1 = \frac{2.0 \ \text{mi}}{7.0 \ \text{min}}* \frac{60 \ \text{min}}{1 \ \text{hr}}vˉ1​=7.0 min2.0 mi​∗1 hr60 min​

vˉ1=d2t2\bar v_1 = \frac{d_2}{t_2}vˉ1​=t2​d2​​

Which for my example problem evaluates to:

vˉ2=4.5 mi14.0 min∗60 min1 hr\bar v_2 = \frac{4.5 \ \text{mi}}{14.0 \ \text{min}}* \frac{60 \ \text{min}}{1 \ \text{hr}}vˉ2​=14.0 min4.5 mi​∗1 hr60 min​
