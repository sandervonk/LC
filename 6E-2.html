Question: Explosives are used to blow a rock apart. The explosion blows the rock into three fragments. Two fragments go off at a 111 degree angle to each other–a 4.2 kg piece at 21.4 m/s and a 3.1 kg piece at 32.8 m/s. Calculate the speed of the third piece, if it has a mass of 4.3 kg.

m1m_1m1​= 4.2 kg
v1v_1v1​ = 21.4 m/s

m2m_2m2​ = 3.1 kg
v2v_2v2​ = 32.8 m/s

m3m_3m3​ = 4.3 kg

θ\thetaθ = 111 degrees

Finding momentums(s)

Core Concepts

We’re solving for: v_3. To do so, we’ll rely on our previous knowledge that an explosion will result in a net momentum of 0 in every dimension (absent gravity et al)

p=0p = 0p=0

so we can apply this to the components of the magnitude to create two new equations

px= 0py= 0\begin{aligned} p_x =& \ 0 \\ p_y =& \ 0 \end{aligned}px​=py​=​ 0 0​

keep these in mind, we’ll be using them after finding the existing momentums!

Finding existing fragment momentums: Fragment 1

since we have no real idea of where this is in space, and it ultimately doesn’t matter to the solution, we can call the direction of the first fragment the x-axis, and θ=0\theta = 0θ=0 (based on the unit circle)

This makes this part fairly easy, since it only has one direction, but we’ll set up both components just to be thorough

Using the momentum equation:
p=m∗vp = m * vp=m∗v
we can create equations for the two. note that θ\thetaθ = 0, and sin⁡(0)=0\sin(0) = 0sin(0)=0, so there is no y-component, or it is equal to 0, however you want to think about it

p1,x= v1∗m1∗cos⁡(θ)p1,y= v1∗m1∗sin⁡(θ)\begin{aligned} p_{1,x} =& \ v_1 * m_1 * \cos(\theta) \\ p_{1,y} =& \ v_1 * m_1 * \sin(\theta) \end{aligned}p1,x​=p1,y​=​ v1​∗m1​∗cos(θ) v1​∗m1​∗sin(θ)​

which at θ=0\theta = 0θ=0 just becomes

p1,x= v1∗m1p1,y= 0\begin{aligned} p_{1,x} =& \ v_1 * m_1 \\ p_{1,y} =& \ 0 \end{aligned}p1,x​=p1,y​=​ v1​∗m1​ 0​

Finding existing fragment momentums: Fragment 2

We can use the equations from the previous question to rewrite write some equations for the second fragment, angled at an angle θ\thetaθ away from the first fragment. We can simply reuse the equations, however now θ\thetaθ is set equal to the value we are given in the problem

p2,x= v2∗m2∗cos⁡(θ)p2,y= v2∗m2∗sin⁡(θ)\begin{aligned} p_{2,x} =& \ v_2 * m_2 * \cos(\theta) \\ p_{2,y} =& \ v_2 * m_2 * \sin(\theta) \end{aligned}p2,x​=p2,y​=​ v2​∗m2​∗cos(θ) v2​∗m2​∗sin(θ)​

Combining existing fragments to find the third

Note: I recommend calculating these values in-between steps to avoid having such long equations

since we know
px= 0py= 0pdirection=∑pdirection,each\begin{aligned} p_x =& \ 0 \\ p_y =& \ 0 \\ p_{direction} =& \sum{p_{direction, each}}\end{aligned}px​=py​=pdirection​=​ 0 0∑pdirection,each​​

we can create new equations and set them equal to their 0 values from before because of what we know about explosions and momentum

0= p1,x+p2,x+p3,x0= p1,y+p2,y+p3,y\begin{aligned} 0 =& \ p_{1,x} + p_{2,x} + p_{3,x} \\ 0 =& \ p_{1,y} + p_{2,y} + p_{3,y} \end{aligned}0=0=​ p1,x​+p2,x​+p3,x​ p1,y​+p2,y​+p3,y​​

now we can substitute in our values from before

0= (v1∗m1) + (v2∗m2∗cos⁡(θ)) + p3,x0= 0 + (v2∗m2∗sin⁡(θ)) + p3,y\begin{aligned} 0 =& \ (v_1 * m_1 ) \ &+ \ (v_2 * m_2 * \cos(\theta)) \ +& \ p_{3, x} \\ 0 =& \ 0 \ &+ \ (v_2 * m_2 * \sin(\theta)) \ +& \ p_{3, y} \end{aligned}0=0=​ (v1​∗m1​)  0 ​+ (v2​∗m2​∗cos(θ)) ++ (v2​∗m2​∗sin(θ)) +​ p3,x​ p3,y​​

now let’s do a bit of expanding and isolate the p3p_3p3​ values

m3∗v3,x=−(v1∗m1) −(v2∗m2∗cos⁡(θ))m3∗v3,y=(v2∗m2∗sin⁡(θ))\begin{aligned} m_3 * v_{3, x}& = -(v_1 * m_1 ) \ - &(v_2 * m_2 * \cos(\theta)) \\ m_3 * v_{3, y}& = &(v_2 * m_2 * \sin(\theta)) \end{aligned}m3​∗v3,x​m3​∗v3,y​​=−(v1​∗m1​) −=​(v2​∗m2​∗cos(θ))(v2​∗m2​∗sin(θ))​

finally, let’s isolate the velocities

v3,x=−(v1∗m1)−(v2∗m2∗cos⁡(θ))m3v3,y=(v2∗m2∗sin⁡(θ))m3\begin{aligned} v_{3, x}& = \frac{-(v_1 * m_1 ) -(v_2 * m_2 * \cos(\theta))}{m_3} \\ v_{3, y}& = \frac{(v_2 * m_2 * \sin(\theta))}{m_3} \end{aligned}v3,x​v3,y​​=m3​−(v1​∗m1​)−(v2​∗m2​∗cos(θ))​=m3​(v2​∗m2​∗sin(θ))​​

We can use the Pythagorean theorem to find speed from the two velocity components, substituting in the values that we

v3= (v3,x)2+(v3,y)2v3= (−(v1∗m1)−(v2∗m2∗cos⁡(θ))m3)2+((v2∗m2∗sin⁡(θ))m3)2\begin{aligned} v_3 =& \ \sqrt{(v_{3,x})^2 + (v_{3,y})^2} \\ v_3 =& \ \sqrt{\left( \frac{-(v_1 * m_1 ) -(v_2 * m_2 * \cos(\theta))}{m_3} \right)^2 + \left(\frac{(v_2 * m_2 * \sin(\theta))}{m_3}\right)^2} \end{aligned}v3​=v3​=​ (v3,x​)2+(v3,y​)2​ (m3​−(v1​∗m1​)−(v2​∗m2​∗cos(θ))​)2+(m3​(v2​∗m2​∗sin(θ))​)2​​

That’s all! your v3v_3v3​ (which is really the magnitude of v3v_3v3​) is the answer you just found above
